### **Lab 1: Класс String с алгоритмом Бойера-Мура**
**Задача:** Реализация собственного класса String с перегрузкой операторов и алгоритмом Бойера-Мура для поиска подстроки.

### **Lab 2: Класс для работы с большими числами (Big Number)**
**Задача:** Реализация класса для работы с большими числами, превышающими стандартные типы данных.

### **Lab 3: Алгоритм Дейкстры для генерации перестановок**
**Задача:**  Реализация алгоритма Дейкстры для генерации всех перестановок массива в лексикографическом порядке.

### **Lab 4: Алгоритм Хаффмана для сжатия данных**
**Задача:** Реализация алгоритма Хаффмана для сжатия и восстановления текстовых данных.

### **Lab 5: Деление на 2**
**Задача:** Реализация деления числа на 2 с использованием битовых операций.
